# ShopWell – Mobile UI Concept (Figma)

ShopWell is a **mobile shopping UI concept** designed entirely in **Figma**, inspired by the clean and minimal design style of the **Nike mobile app**. This project focuses on **visual design, layout structure, and mobile UI components**, without any interactive prototyping.

---

##  **Project Overview**

This mobile design includes:

* A clean home screen with hero banner
* Product cards featuring images, pricing, and descriptions
* Simple top navigation structure
* Minimalistic color palette and bold typography
* A layout inspired by modern retail mobile apps

The goal of the project was to practice **mobile UI design**, spacing, alignment, and consistent component usage.

---

##  **Design Highlights**

###  Built 100% in Figma

* Auto layout
* Components
* Grids & spacing
* Iconography
* Modern mobile UI patterns

###  Nike-Inspired Aesthetic

* Large imagery
* Clean typography
* Simple card layout
* High contrast areas

###  Static UI (No Prototype)

This project focuses on **visual design only**, without navigation flows or interactions.
